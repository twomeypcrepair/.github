## What lives here

This GitHub organization hosts the **bare scripts** behind some of the tools we publish for our customers and our own shop workflow.

Most of these tools are also available as **signed, ready-to-use compiled `.exe` versions** through our [public Toolbox](https://twomeypcrepair.com/toolbox/) — those are the easiest way to use them.

## What we share here vs. what's compiled

|  | What's published here | What's in the signed `.exe` |
|---|---|---|
| **Code** | Just the bare script of the function being shared | Full source compiled and signed |
| **Use** | Read it, reference it, learn from it | Download and run |
| **Branding** | Plain code | Compiled tools display a small footer credit linking back to twomeypcrepair.com |

If you want to **use** the tool, grab the signed `.exe` from the [Toolbox](https://twomeypcrepair.com/toolbox/). If you want to see what a piece of it does **under the hood**, the raw script is here.

## About our compiled tools

- **Signed.** Every Windows binary we release is signed with our verified code-signing certificate.
- **No adware. No sponsor offers. No bundled junk.**
- **Small footer credit.** Compiled tools display a small footer linking back to twomeypcrepair.com — that's how new users find the rest of our toolbox.
