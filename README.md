## Who we are

Twomey PC Repair is a one-shop independent tech repair business in Highland Square Shopping Center, serving north-central Arkansas — Hardy, Cherokee Village, Ash Flat, Mammoth Spring, Salem, Pocahontas, Mountain Home and beyond.

We fix **phones, computers, game consoles, tablets, and networks**. We build **custom gaming PCs and business networks**. We design **websites and run social media** for local businesses. We charge by **time spent**, not flat service fees.

## What lives here

This GitHub organization hosts the **open-source tools, scripts, and installers** we build for our customers and our own shop workflow. Most of these are also distributed signed and ready-to-use through our [public Toolbox](https://twomeypcrepair.com/toolbox/).

| Project | What it does | Status |
|---|---|---|
| **Twomey Installer** | One-click signed installer for 179+ free Windows apps. Pick what you want, click download, get a custom signed `.exe`. No adware, no junk, ever. | 🟢 Active |
| **PowerShell Scripts** | Tune-ups, driver discovery, Windows optimization, malware cleanup, and other diagnostic and repair scripts we use daily on the bench. | 🟢 Active |
| **Batch Scripts** | Quick fixes — print spooler resets, network troubleshooting, log collection, and other one-shot Windows fixes. | 🟢 Active |
| **3D Printable Adapters** | STL files for SSD bay adapters, mounting brackets, and replacement parts we 3D-print in-shop. | 🟡 Slow drip |
| **Twomey Website** | Public-facing site stack (WordPress + Bricks + custom plugins) — most of our internal tooling lives in repos here. | 🔵 Internal |

## Our promises about everything we publish

- **Signed.** Every Windows binary we release is signed with our verified code-signing certificate. If your browser or Windows warns about the publisher, the cert chain should resolve back to "Twomey PC Repair."
- **No adware. No sponsor offers. No "free" software with toolbar surprises.** If we ever take an upstream tool and notice it bundles junk, we strip it and explain in the changelog.
- **Curated.** We use what we publish. If a tool stops being useful, we archive it.
- **Updated weekly.** Most repos see updates at least weekly during active development.

## How to use these projects

**Most users:** download our pre-built signed releases from the [Toolbox](https://twomeypcrepair.com/toolbox/) — easier than cloning and building.

**Tinkerers:** clone the repo, read the README, build/run from source. Issues and pull requests welcome.

**Other shop owners / techs:** feel free to fork anything for your own use. If you find a bug or want to contribute, open a PR. We'll respond.
